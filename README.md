# get_amap_poi_plus
只需要浏览器，就能下载高德地图的poi数据。<br>
用了javascript和 jQuery。<br>
还有<a href="https://github.com/rtyfghvbnlndl/get_amap_POI_data">Python版链接</a>，下载大量数据时会更好用。

### 使用方法 
1. 下载pagecopy.htm
2. 用浏览器打开
